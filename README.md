Ice Cream – Passion in Every Scoop 🍦
A modern, vibrant and fully responsive website for a premium ice cream brand. Built with pure HTML & CSS.

![Website Preview](https://ice-web.surge.sh/)

✨ Features
Eye-catching hero section with passionate branding
Beautiful product showcase (Ice Cream, Ice Coffee, Milkshakes)
"How It's Made" section with tradition & love story
Customer testimonial slider
Multiple location cards (Cafe + Food Truck)
Fully responsive design
Modern pink & pastel color scheme
🎨 Design Highlights
Custom typography using DM Sans and Titan One
Smooth hover effects
Beautiful product cards with floating images
Warm and playful ice cream theme
Professional footer with location details
🛠️ Technologies Used
HTML5
CSS3 (Flexbox & Grid)
Google Fonts
Font Awesome Icons
Fully Responsive Layout
